Instructions

  1. Click on the buttons OR Use the Keyboard
  2. Number buttons correspond to buttons
  3. Maths: '+', '-', '*', '/' correspond to keyboard buttons
  4. Decimal corresponds to decimal button
  5. For Equal to, = use Shift
  6. For Backspace, use Backspace
  7. For clearing, C use \ (also Pipe Button on most keyboards)
  8. Report errors or give feedback at MS.bin.gmail.com
